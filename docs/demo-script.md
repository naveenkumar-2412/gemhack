# Demo Script (3m 30s)

## 0:00-0:20 — Problem
"Most assistants are still text boxes. TriSense is a live multimodal agent that can hear, see, and create mixed-media outputs in real time."

## 0:20-1:30 — Live Agent
- Select **Live Agent** mode.
- Enter or speak a request with a follow-up interruption.
- Show quick response and `Speak Last Response` output.
- Highlight interruption-ready architecture with WebSocket + short responses.

## 1:30-2:30 — Creative Storyteller
- Switch to **Creative Storyteller**.
- Prompt: "Create a 15-second product launch concept."
- Show interleaved output: story text + image prompt + voiceover + shot list.
- Explain this is one cohesive multimodal response package.

## 2:30-3:10 — UI Navigator
- Switch to **UI Navigator**.
- Provide a user intent and screenshot context.
- Show returned actionable plan and confidence value.

## 3:10-3:30 — Cloud Proof + Close
- Open Google Cloud Run service and logs.
- Show Firestore session records.
- Close with value: one platform, three multimodal experiences.
